
# 🤝 贡献指南

感谢你对 **IntuitionMath** 的关注！我们致力于构建全球最优雅的数学直觉引擎。

## 🧪 内容准则：3Blue1Brown 标准

新增任何内容时，请自问以下三个问题：
1. **直觉先行**：是否通过几何图像先建立了直觉，而非直接抛出公式？
2. **逻辑连贯**：推导步骤之间是否存在逻辑跳跃？
3. **视觉回馈**：当用户调节参数时，图像是否能清晰反映数学内涵的变化？

## 🛠️ 开发工作流

1. **定义主题 ID**：在 `types.ts` 的 `MathTopic` 枚举中新增标识符。
2. **编写故事线**：在 `constants.ts` 中添加 `TOPIC_REGISTRY` 条目。
3. **实现渲染逻辑**：在 `VisualPanel.tsx` 的 `RENDER_STRATEGIES` 中编写对应的 D3 绘制逻辑。
4. **注入 AI 策略**：为该主题配置专属的 `aiSystemPrompt`。

## 💎 代码风格
- 优先使用 React 函数式组件。
- 严格遵循 TypeScript 类型定义。
- 视觉渲染代码需保持高性能，避免在 `useEffect` 中进行不必要的重计算。
